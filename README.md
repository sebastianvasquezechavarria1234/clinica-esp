# 🏥 Clínica ESP — Plataforma de Salud Holística
**React 18** · **Vite** · **Tailwind CSS** · **React Router**

🌟 **¿Qué es este proyecto?**
Este repositorio contiene la migración y modernización de la plataforma web de **Clínica ESP**. Originalmente construido con HTML/CSS/JS tradicional, el proyecto ha sido transformado en una **Single Page Application (SPA)** moderna para ofrecer una experiencia premium y escalable.

El núcleo de la aplicación actual se encuentra en la carpeta `clinica-react`.

✨ **Características Principales**
*   🚀 **Navegación Instantánea**: SPA con React Router para transiciones fluidas.
*   📱 **Diseño Adaptativo**: Experiencia móvil optimizada con Tailwind CSS.
*   🎨 **Rich Aesthetics**: Interfaz premium con animaciones y sistema de diseño curado.
*   📅 **Gestión de Citas**: Flujo dinámico para la reserva de servicios médicos.
*   📖 **Blog Informativo**: Artículos integrados sobre salud y bienestar holístico.

🛠️ **Stack Tecnológico**
| Capa | Tecnologías |
| :--- | :--- |
| **Framework** | React 18 |
| **Herramienta de Construcción** | Vite |
| **Estilos** | Tailwind CSS + PostCSS |
| **Enrutamiento** | React Router Dom v6 |

🚀 **Instalación y Uso Local**
1. **Entrar en la carpeta del proyecto React**
   ```bash
   cd clinica-react
   ```
2. **Instalar dependencias**
   ```bash
   npm install
   ```
3. **Ejecutar el servidor de desarrollo**
   ```bash
   npm run dev
   ```
   Accede a la app en `http://localhost:5173`.

📁 **Estructura del Repositorio**
*   `/clinica-react`: **Proyecto principal (React + Vite)**.
*   `/css`, `/js`, `/img`: Archivos del proyecto estático original (Legacy).
*   `*.html`: Páginas originales (Legacy).
